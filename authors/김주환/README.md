# 김주환

김주환 교수님의 유튜브 강의 자막을 주제별로 정리한 아카이브입니다.

- `urls.txt` — 다운로드 대상 유튜브 영상 URL 목록
- `failed_urls.txt` — 자막 다운로드에 실패한 영상 목록 (자막이 없거나 오류, 있을 때만 생성됨)
- `subtitles/` — 주제별로 분류된 자막 TXT 파일 (총 167개)

`12_기타`에 새 파일이 쌓이면 그 파일들의 제목을 보고 필요한 키워드를 `CATEGORIES`에 추가한 뒤
`organize_subtitles.py`를 재실행해 재분류할 수 있습니다.
